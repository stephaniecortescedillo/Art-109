Calorie Archive

This archive was created with the idea of good nutrition in mind. The information is correct, but simplified. Omitting things like whether the food is healthy or not, and reducing meals to their calorie count. This is done to show how, even leaving out important details, information can still create an impact. The graph shows the time difference between meals.
